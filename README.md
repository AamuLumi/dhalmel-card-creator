# MeilisearchMate

Simple webapp to debug Meilisearch  
Built with React + Typescript  
Build tool : Vite / SWC  
Components : NextUI  
CSS : Tailwind

## Installation

```
> npm i
```

## Run in dev mode

```
> npm run dev
```

## Build for production

Default output folder is `<project_folder>/dist`.

```
> npm run build
```
